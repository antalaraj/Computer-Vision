# Image Classification Projects using CNN & Transfer Learning

This repository contains two deep learning projects focused on **image classification** using Convolutional Neural Networks (CNN) and **Transfer Learning** with VGG16. Both projects are implemented in Python using TensorFlow and Keras.

These projects demonstrate practical skills in:
- Data preprocessing
- Model building
- Training & evaluation
- Performance improvement using deep learning techniques

---

## Projects Included

### 1. CIFAR-10 Image Classification with CNN
**File:** `CIFAR-10 Image Classification with CNN.ipynb`

A custom-built CNN model trained on the **CIFAR-10 dataset**, which consists of 60,000 color images across 10 classes.

#### Key Features:
- Custom CNN architecture
- Data normalization and augmentation
- Achieved high validation accuracy
- Visualization of training performance

#### Classes:
- Airplane  
- Automobile  
- Bird  
- Cat  
- Deer  
- Dog  
- Frog  
- Horse  
- Ship  
- Truck  

---

### 2. Cats vs Dogs Classification using Transfer Learning (VGG16)
**File:** `Cats_vs_Dogs_Transfer_Learning_VGG16.ipynb`

An image classification model using **VGG16 pre-trained on ImageNet** to classify images of cats and dogs.

#### Key Features:
- Transfer learning with VGG16
- Feature extraction + fine-tuning
- Data augmentation
- High accuracy with fewer training samples
